# Switch

A control that allows the user to toggle between checked and not checked.

You may find all of the api docs there [Documentation](https://www.radix-ui.com/primitives/docs/components/switch)

## Installation

<Source code={`npm i @giosg/design-system-switch`} language="bash" dark />

## Usage

<Subtitle>Basic example</Subtitle>
<Source code={`
import {Switch} from "@giosg/design-system-switch";

export const Base = () => {
    return <Switch />;
  },
};
`}  dark/>

<Subtitle>Manually-assembled example</Subtitle>
<Source code={`
import {_Root, _Thumb} from "@giosg/design-system-switch";

export const ManuallyAssembled = () => {
    return (
      <_Root>
        <_Thumb/>
      </_Root>
    );
  },
};
`}  dark/>
